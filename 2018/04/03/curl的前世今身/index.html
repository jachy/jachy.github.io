curl的前世今身

发表于 2018-04-03

What is curl？

command line tool and library for transferring data with URLs
curl是一个利用URL语法在命令行下工作的文件综合传输工具，支持文件下载和上传

Supports…

CURL支持的通信协议有FTP、FTPS、HTTP、HTTPS、TFTP、SFTP、Gopher、SCP、Telnet、DICT、FILE、LDAP、LDAPS、IMAP、POP3、SMTP和RTSP

Who makes curl?

来自瑞典（斯德哥尔摩）的Daniel Stenberg 于1996年末
基于开源工具HttpGet二次开发的

What’s the latest curl?

最新版本：7.54.1, 发行于2017年6月14日
我们目前线上的版本：7.54.0

SYNOPSIS

Options

-o < file_name > 保存为file_name的文件
-e ‘referer_name’ 指定referer
-x, –proxy [protocol://]host[:port] 指定某主机的某个端口，一般用于http
–resolve < host:port:address > 指定某主机的某个端口，一般用于https
–connect-timeout < seconds > 指定超时时间
-Y, –speed-limit < speed > 下载真实速度低于这个value，就断开
-y, –speed-time < seconds > 下载真实时间低于这个value，就断开
-H, –header < header > 指定访问的头部
-A, –user-agent < name > 指定UA
-v, –verbose
-V, –version
–limit-rate < speed > 指定下载速度
-m, –max-time < time > 指定下载超时时间
-L, –location 拉取跳转后的内容
-w, –write-out < format > 以什么格式输出
-r, –range 分段下载，也叫请求range
-u 用户名:密码
-T < file_name >上传文件

Common usage

1.测试referer访问

2.测试https访问

3.测试限制下载速度访问

4.获取302后的内容

5.显示下载大小，下载速度等

6.下载速度为2k，超时时间为2s

7.模拟range请求或者分段下载

7.指定UA
